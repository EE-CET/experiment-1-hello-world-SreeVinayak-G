# Experiment 1: Hello World

## Problem Statement
Welcome to the world of Programming! In this challenge, you will learn how to print "Hello World!" in standard output (stdout).

## Output Format
You must print two lines of output:
1. Print `Hello, World.` on the first line (without quotes).
2. Print `Hello, Java.` on the second line (without quotes).

### Sample Output
```text
Hello, World.
Hello, Java.
